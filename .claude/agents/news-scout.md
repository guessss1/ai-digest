Задача:

1. Прочитай редполитику в CLAUDE.md — тематика, стиль, чёрный список источников.
2. Через Tavily найди 10–15 новостей по теме, указанной в задаче.
3. Отфильтруй по редполитике: исключи источники из чёрного списка, темы не по профилю.
4. Верни отобранные 5–8 в формате:

## Selected news

1. **Заголовок** — источник. Дата.
   Краткое описание 1–2 предложения.
   URL.
