Китайская лаборатория DeepSeek выложила preview семейства V4. В релизе две версии: V4 Flash для массовых задач и V4 Pro для сложных рассуждений. Обе доступны через API и в виде весов на Hugging Face — лаборатория продолжает практику открытых релизов после успеха V3 и R1.

По заявленным бенчмаркам V4 Pro обходит open-source конкурентов в агентных сценариях: использование инструментов, многошаговое планирование, навигация по кодовой базе. На SWE-bench Verified модель показывает результат, сопоставимый с проприетарными системами среднего класса. В knowledge-тестах вроде MMLU-Pro и GPQA V4 Pro пока уступает GPT-5.5 и Gemini 3.1 Pro, но разрыв сократился до нескольких процентных пунктов.

Главный аргумент DeepSeek — цена. V4 Flash обойдётся примерно в десять раз дешевле сопоставимых frontier-моделей за миллион токенов. V4 Pro дороже, но всё ещё в разы выгоднее GPT-5.5 при близких показателях на агентных задачах. Лаборатория объясняет экономику архитектурными решениями: смесь экспертов с агрессивным sparsity, обучение на собственном кластере H800, оптимизированный inference-стек.

Для разработчиков это означает несколько практических следствий. Во-первых, появилась open-weight база, на которой можно строить агентные пайплайны без оглядки на OpenAI или Anthropic. Во-вторых, цена API делает осмысленным переход на DeepSeek для длинных контекстов и фоновых задач — суммаризации, индексации, batch-обработки документов. В-третьих, веса можно развернуть локально, если требуется data residency или офлайн-режим.

Контекстное окно V4 Pro заявлено в 256k токенов, V4 Flash — 128k. Обе модели поддерживают function calling в формате, совместимом с OpenAI SDK. Это упрощает миграцию: достаточно поменять base URL и ключ. Структурированный вывод через JSON Schema поддерживается нативно.

Открытые вопросы остаются. Lab-релиз — это preview, без гарантий стабильности API. Веса опубликованы под лицензией DeepSeek License, которая разрешает коммерческое использование, но содержит ограничения по геораспространению. Бенчмарки приведены самой лабораторией; независимые замеры от LMSys и Artificial Analysis ожидаются в ближайшие недели.

Релиз продолжает тренд 2026 года: фронтир-уровень становится коммодити, а ценовое давление на закрытые лаборатории растёт. Если независимые тесты подтвердят заявленные цифры, V4 станет дефолтным выбором для агентных продуктов с ограниченным бюджетом.
