## Claude Opus 4.7: high-res vision и первое место на SWE-bench Pro

16 апреля Anthropic выпустила Claude Opus 4.7 — и это не просто косметический апдейт. Главное: модель стала первой в линейке Claude с поддержкой изображений до 2576 px (~3.75 MP), что открывает применение в задачах с детализированными схемами, скриншотами интерфейсов и техническими чертежами.

По бенчмаркам: 87.6% на SWE-bench Verified и 64.3% на SWE-bench Pro — первое место среди всех GA-моделей. Добавлен режим `xhigh` effort для тонкой настройки глубины рассуждений, когда стандартного режима недостаточно. Цена осталась прежней: $5/$25 за миллион токенов. Доступно через API (`claude-opus-4-7`), Bedrock, Vertex AI и Microsoft Foundry. Отдельно запустилась программа Cyber Verification Program — верификация для легитимных задач в кибербезопасности.

## OpenAI Agents SDK v0.14: изолированные воркспейсы для агентов

15 апреля в PyPI вышел `openai-agents-python` v0.14.0 с концепцией Sandbox Agents. Агент теперь работает в изолированном контейнере с workspace manifests — декларативным описанием окружения. Снапшоты и resume позволяют продолжить работу между сессиями без потери контекста.

В v0.14.2 (18 апреля) добавлен MongoDB-бэкенд для хранения сессий, в v0.14.3 — Windows CI. TypeScript-поддержка пока в планах. Все функции работают через стандартный API без дополнительной наценки. По данным TechCrunch, Oscar Health уже использует SDK для автоматизации клинических записей в продакшне.

## Codex App 26.415: Computer Use, браузер и память

16 апреля вышел Codex App 26.415 с Computer Use: агент управляет macOS-приложениями через собственный курсор, без Accessibility API. Встроенный браузер позволяет итерировать фронтенд прямо в приложении. Memories сохраняют предпочтения между сессиями, Automations планируют задачи на будущее.

Дополнительно: интеграция с GitHub PR review, генерация изображений через `gpt-image-1.5`, 90+ новых плагинов (MCP + навыки), поддержка SSH для удалённых devbox. Codex CLI 0.121.0 (15 апреля) добавил marketplace и realtime API с мультимодальным выводом.

## xAI Speech API: STT на 25 языках и TTS для голосовых агентов

18 апреля xAI открыл standalone Speech-to-Text и Text-to-Speech API — тот же стек, что работает внутри Grok Voice и Tesla Infotainment. STT поддерживает 25+ языков, 12 форматов аудио, word-level timestamps и speaker diarization. Два режима: real-time WebSocket streaming и batch REST. Batch API теперь умеет генерацию изображений, видео и client-side function tools в одном запросе. Тот же стек обслуживает Starlink customer support — то есть нагрузку реальная.

## GitHub Copilot заморозил новые подписки Pro/Pro+

20 апреля GitHub приостановил регистрации на планы Pro, Pro+ и Student — Free остался открытым. Из Pro убраны Opus-модели, Opus 4.7 теперь только на Pro+. Существующие подписчики, которых не устраивают изменения, могут запросить возврат до 20 мая 2026.

Из хорошего: Auto model selection — автовыбор модели по задаче — стал GA для всех планов в Copilot CLI. Предупреждения о лимитах теперь видны прямо в VS Code и CLI, а не только в настройках аккаунта.
