На Cloud Next '26 Google свела апдейты по AI-стеку в одну историю: платформа для корпоративных агентов, расширение модельного портфеля и цифры по нагрузке.

Центральный анонс — Gemini Enterprise Agent Platform. Это не просто SDK, а законченная среда для жизненного цикла агента: дизайнер сценариев, реестр инструментов, observability, политика доступа к данным. Платформа ставит MCP в основу подключения внешних источников и поддерживает A2A-протокол для коммуникации между агентами разных вендоров. По сути, Google пытается занять место «AWS для агентов»: предоставить инфраструктурный слой, чтобы корпоративные команды не собирали оркестрацию вручную.

Второе важное движение — расширение Vertex AI Model Garden. Claude Opus 4.7 теперь доступен в Google Cloud наряду с Gemini 3.1 Pro и линейкой open-weight моделей. Это сигнал: Google готова предлагать клиентам мультивендорный портфель, не настаивая на собственных моделях. Для разработчиков это означает единый IAM, единый биллинг и общий уровень безопасности при работе с разными провайдерами.

Цифра, которую цитируют все обзоры — 16 миллиардов токенов в минуту через прямой API. В прошлом квартале было 10 миллиардов. Рост на 60% за квартал говорит, что инференс становится массовой нагрузкой на уровне CDN или баз данных, а не экспериментом. Google использует эту цифру, чтобы продавать TPU-инфраструктуру: показано, что новая итерация TPU и Pathways позволяет держать такой объём с приемлемой латентностью.

Из менее заметного, но практичного:

- Vertex AI Agent Builder получил визуальный конструктор пайплайнов с импортом из LangGraph.
- BigQuery теперь умеет вызывать модели Gemini напрямую в SQL без ML.PREDICT-обвязки.
- AlloyDB добавил векторный поиск на базе SCANN с автоматическим выбором индекса.
- Cloud Run получил профиль GPU с шаринговой моделью оплаты для inference-нагрузок.

Что это значит для команд. Если стек уже на Google Cloud, миграция агентов на Enterprise Agent Platform станет дефолтным путём — особенно ради observability и compliance. Если стек мультиоблачный, сам факт включения Claude Opus 4.7 в Vertex снижает блокировку и упрощает A/B-тесты между провайдерами. Цена и лимиты Agent Platform пока не опубликованы; Google обещает прайс-лист в течение месяца.
