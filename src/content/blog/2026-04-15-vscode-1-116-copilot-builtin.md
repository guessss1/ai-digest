15 апреля Microsoft выпустила VS Code 1.116. Главное изменение — GitHub Copilot Chat стал встроенным расширением: отдельно ставить его больше не нужно, отключить можно в настройках. Команда объясняет решение тем, что Chat превратился в основной интерфейс к агентской функциональности, и разделение на отдельные плагины запутывало пользователей. Для тех, кто сидел на сторонних AI-ассистентах, ничего не меняется — встроенный Copilot можно выключить, маркетплейс по-прежнему открыт.

Из другого заметного: Agent Debug Logs — теперь видно, что делал агент в предыдущих сессиях, включая полную историю вызовов инструментов, аргументы и возвращённые значения. Удобно разбираться, когда агент ушёл не туда: можно прокрутить последовательность решений и понять, на каком шаге потерялся контекст или где инструмент вернул неожиданный результат. Copilot CLI получил параметр thinking effort для явного управления глубиной размышлений — полезно, когда хочется экономить токены на простых задачах или, наоборот, дать модели больше времени на сложный рефакторинг. Ещё добавили JS/TS Chat Features в preview — специализированные скиллы для TypeScript и JavaScript: работа с tsconfig, автоматический разбор типовых ошибок компилятора, подсказки по настройке сборки.

В Insiders-ветке 1.117 параллельно катят сабагенты и три режима автосогласования. Default оставляет ручное подтверждение на каждом действии — классика. Bypass пропускает низкорисковые операции (чтение файлов, запуск тестов) и спрашивает только на записи и shell-командах. Autopilot работает полностью автономно, только логирует — режим для фоновых агентов, которые выполняют задачу от начала до конца. Добавили изоляцию через git worktree: один агент запускает другого в отдельном рабочем дереве, без конфликтов по файлам и ветке, что открывает дорогу к параллельным сабагентам с разными гипотезами решения.

Сдвиг понятный: VS Code догоняет Cursor и Claude Code по модели «не один ассистент, а парк». Cursor 3 в начале апреля собрал похожий интерфейс с Agents Window и параллельным запуском до восьми агентов. Разница скорее во вкусе и встроенности — Copilot-экосистема срослась с GitHub и PR-флоу, Cursor делает ставку на собственные модели и редактор как продукт, Claude Code остаётся CLI-first. Выбор между ними теперь ближе к выбору IDE, чем к выбору плагина.
