15 апреля Microsoft выпустила VS Code 1.116. Главное изменение — GitHub Copilot Chat стал встроенным расширением: отдельно ставить его больше не нужно, отключить можно в настройках. Команда объясняет решение тем, что Chat превратился в основной интерфейс к агентской функциональности, и разделение на отдельные плагины запутывало пользователей.

Из другого заметного: Agent Debug Logs — теперь видно, что делал агент в предыдущих сессиях, включая полную историю вызовов инструментов. Удобно разбираться, когда агент ушёл не туда. Copilot CLI получил параметр thinking effort для явного управления глубиной размышлений. Ещё добавили JS/TS Chat Features в preview — специализированные скиллы для TypeScript и JavaScript.

В Insiders-ветке 1.117 параллельно катят сабагенты и три режима автосогласования: Default (ручное подтверждение), Bypass (автоматический пропуск низкорисковых операций), Autopilot (полная автономность). Добавили изоляцию через git worktree — один агент запускает другого в отдельном рабочем дереве, без конфликтов по файлам и ветке.
