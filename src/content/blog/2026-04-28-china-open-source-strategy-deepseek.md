The New York Times опубликовал разбор стратегии китайских AI-компаний на фоне релиза DeepSeek V4. Главный тезис материала: open-source перестал быть техническим решением и стал инструментом борьбы за глобальный рынок.

## Контекст

Ровно год назад DeepSeek V3 встряхнул индустрию — модель на уровне GPT-4 вышла под почти-открытой лицензией и обошлась в обучении на порядок дешевле западных аналогов. После этого крупные китайские игроки — Alibaba с Qwen, Baidu с Ernie, Moonshot с Kimi — ускорили публикацию весов своих frontier-моделей.

V4 продолжает этот курс и доводит его до логического конца: чистая MIT-лицензия, никаких ограничений на коммерческое использование, веса доступны через Hugging Face и зеркала.

## Зачем открывать веса

NYT приводит несколько мотивов, которые называют сами китайские компании и аналитики:

- Захват разработчиков в странах, где американские API недоступны или дороги.
- Снижение зависимости от западных поставщиков инфраструктуры.
- Демонстрация технологического паритета без необходимости проходить экспортные согласования.
- Давление на бизнес-модель проприетарных лабораторий — OpenAI и Anthropic сложнее продавать API, когда сопоставимые веса лежат публично.

Открытость сама по себе не приносит денег, но создаёт экосистему. Дообучения, форки, специализированные дистилляты — всё это формирует долгосрочную привязку к материнской модели.

## Реакция на Западе

В США позиция расколота. Часть индустрии — Meta с Llama, Mistral в Европе — давно работает в open-weight парадигме. Закрытые лаборатории отвечают сдвигом в сторону агентных продуктов: API и ChatGPT-подобные интерфейсы сложнее воспроизвести через выложенные веса.

Регуляторы обсуждают ограничения на скачивание моделей китайского происхождения для критической инфраструктуры. Конкретных запретов на момент публикации NYT не введено.

## Что это значит для разработчиков

Расширяется выбор моделей, которые можно развернуть on-prem без лицензионных рисков. Проверка происхождения весов и аудит обучающих данных становятся частью стандартного процесса при выборе модели для продакшна. Для российских и европейских команд открытые китайские модели — рабочий вариант, когда доступ к API из США затруднён.

Стратегический сдвиг описан в статье как долгосрочный: китайские компании планируют ежеквартальные релизы открытых frontier-моделей минимум до 2027 года.
